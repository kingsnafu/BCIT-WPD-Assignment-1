=== WooCommerce Product Addons ===

WooCommerce Bulk Variations allows your shoppers to add more than one variation at a time to the cart.  Great for wholesales, B2B sites, and for easing the process of adding more than one variation at a time for anyone. 

== Installation ==

1. Upload the folder 'woocommerce-bulk-variations' to the '/wp-content/plugins/' directory

2. Activate 'WooCommerce Bulk Variations' through the 'Plugins' menu in WordPress

== Usage ==

1. Edit or create a Variable Product with exactly two variation attributes. 

2. From the product edit screen look for the input box "Bulk Variation Input"

3. Choose the attribute to use as the Row and as the Column

4. Save the product. 

5. Users will now see a button on the store to switch to the bulk variation input form.
